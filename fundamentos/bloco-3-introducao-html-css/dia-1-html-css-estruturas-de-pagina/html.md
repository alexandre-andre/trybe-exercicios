# HTML - Hyper-text Markup Language
HTML é uma linguagem de marcação de texto com o objetivo de compartilhar informação.

# Estrutura das tags
```html
<nomeTag atributo="valor">conteúdo</nomeTag>
```

# Configuração básica
```html
<!DOCTYPE html>
<html lang="pt-br">
    <head>
        <meta charset="UTF-8">
        <title>Título na aba</title>
    </head>
    <body>
        <h1>título</h1>
        <h2>sub-título</h2>
        <p>
            parágrafo
        </p>
    </body>
</html>
```
